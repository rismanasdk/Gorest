# GoRest API CLI Tool
A simple Python script to interact with GoRest API.

## How to use:
1. Get your token from [GoRest Official Website](https://gorest.co.in/).
2. Login (using GitHub/Google/Microsoft) -> Copy your Access Token from the **Access Token** section.
3. Open `gorest.py` in your code editor.
4. Replace `<TOKEN_GOREST>` with your actual secret token.
5. Run the script: 
   ```bash
   python gorest.py
